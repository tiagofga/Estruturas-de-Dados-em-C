# Stack — Sequential LIFO Stack

[Português](../Pilha.md) | **English**

## Overview

Implementation of a sequential stack in C. A stack follows the **LIFO** principle: last in, first out.

The most recently inserted element is the first one removed.

---

## Main operations

- create and destroy the stack;
- push an element;
- pop an element;
- inspect the top element;
- check whether the stack is empty or full;
- inspect the current size.

---

## Complexity summary

| Operation | Complexity |
|-----------|------------|
| Push | O(1) |
| Pop | O(1) |
| Peek top | O(1) |
| Empty/full check | O(1) |

---

## Use cases

Stacks are commonly used in:

- function call management;
- expression evaluation;
- undo/redo systems;
- depth-first search (DFS);
- syntax parsing.
