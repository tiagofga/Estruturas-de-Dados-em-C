# Sorting Methods for Arrays

[Português](../MetodosOrdenacao.md) | **English**

## Overview

This module implements classic sorting algorithms for arrays in C. The goal is educational: compare implementation strategies, time complexity, and algorithmic behavior.

---

## Implemented methods

| Method | Best | Average | Worst | Stable |
|--------|------|---------|-------|--------|
| Bubble Sort | O(n) | O(n²) | O(n²) | Yes |
| Insertion Sort | O(n) | O(n²) | O(n²) | Yes |
| Selection Sort | O(n²) | O(n²) | O(n²) | No |
| Merge Sort | O(n log n) | O(n log n) | O(n log n) | Yes |
| Quick Sort | O(n log n) | O(n log n) | O(n²) | No |
| Heap Sort | O(n log n) | O(n log n) | O(n log n) | No |

---

## Notes

Simple quadratic algorithms are useful for understanding basic sorting mechanics. Divide-and-conquer and heap-based methods are more suitable for larger inputs.
