# Lista

Implementação de **lista sequencial dinâmica** em C.

## Operações implementadas
- criar e destruir lista
- verificar se está vazia
- inserir no fim
- inserir em posição específica
- remover por posição
- buscar valor
- imprimir elementos

## Executar
```bash
cd /tmp/workspace/tiagofga/Estruturas-de-Dados-em-C/Lista
make
make run
```
