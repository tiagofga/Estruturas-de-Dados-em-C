# Exercícios 02 — Pilhas e Filas

## Pilhas

1. Use uma pilha para inverter uma sequência de inteiros.
2. Implemente uma função que verifica parênteses balanceados em uma string.
3. Implemente uma pilha dinâmica encadeada.
4. Compare pilha estática, pilha dinâmica com vetor e pilha encadeada.

## Filas

1. Simule uma fila de atendimento com prioridade simples.
2. Implemente uma fila dinâmica encadeada.
3. Implemente uma fila circular que redimensiona automaticamente.
4. Use uma fila para simular BFS em um grafo simples.

## Questões de análise

1. Por que pilhas são úteis em chamadas recursivas?
2. Em quais casos uma fila circular evita movimentações desnecessárias?
3. Qual é a diferença conceitual entre fila comum e fila de prioridade?
