# Exercícios — Estruturas de Dados e Algoritmos em C

Esta pasta reúne exercícios práticos por módulo. A proposta é usar os exercícios como extensão das implementações existentes, reforçando análise de complexidade, testes e comparação entre estruturas.

## Módulos

| Arquivo | Tema |
|--------|------|
| [01-listas.md](./01-listas.md) | Lista sequencial, lista encadeada e lista duplamente encadeada |
| [02-pilhas-filas.md](./02-pilhas-filas.md) | Pilhas e filas |
| [03-hash-heap.md](./03-hash-heap.md) | Tabela hash, heap e fila de prioridade |
| [04-arvores.md](./04-arvores.md) | BST e AVL |
| [05-grafos.md](./05-grafos.md) | Grafos com matriz e lista de adjacência |
| [06-busca-ordenacao-complexidade.md](./06-busca-ordenacao-complexidade.md) | Busca, ordenação e análise assintótica |

## Sugestão de uso

1. Leia a documentação do módulo.
2. Compile e execute o exemplo com `make run`.
3. Execute os testes com `make test`.
4. Resolva os exercícios criando novos testes em `tests/`.
5. Compare a complexidade esperada com a implementação criada.
