# Exercícios 05 — Grafos

## Matriz de adjacência

1. Implemente uma função que calcula o grau de um vértice em grafo não-direcionado.
2. Implemente detecção de ciclo em grafo direcionado.
3. Compare o uso de memória com lista de adjacência para grafos esparsos.
4. Adicione testes para grafo desconexo.

## Lista de adjacência

1. Implemente contagem de arestas.
2. Implemente Prim para árvore geradora mínima.
3. Implemente Bellman-Ford para pesos negativos.
4. Implemente busca de componentes conexas.

## Questões de análise

1. Quando matriz de adjacência é melhor que lista de adjacência?
2. Qual é a complexidade de listar vizinhos em cada representação?
3. Por que Dijkstra não deve ser usado com pesos negativos?
