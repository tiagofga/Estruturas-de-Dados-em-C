# Exercícios 03 — Tabela Hash, Heap e Fila de Prioridade

## Tabela Hash

1. Implemente redimensionamento automático quando o fator de carga passar de 0,75.
2. Implemente uma função que retorna o maior comprimento de cadeia nos baldes.
3. Compare colisões usando diferentes capacidades: 5, 7, 10 e 101.
4. Adapte a tabela para armazenar strings como chaves.

## Heap

1. Implemente um min heap com a mesma interface do max heap.
2. Implemente uma fila de prioridade usando pares `(prioridade, valor)`.
3. Use heap para obter os `k` maiores elementos de um vetor.
4. Compare heap sort com quick sort e merge sort para diferentes tamanhos.

## Questões de análise

1. Quando uma tabela hash pode degradar para O(n)?
2. Por que heap não permite busca eficiente por valor arbitrário?
3. Qual é a diferença entre árvore binária de busca e heap?
