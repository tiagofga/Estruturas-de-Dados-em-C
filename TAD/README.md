# Tipo Abstrato de Dados (TAD)

TAD define **o que** uma estrutura faz (operações e regras), sem expor **como** ela é implementada.

Neste repositório:
- os arquivos em `include/` representam a interface pública (TAD)
- os arquivos em `src/` representam a implementação interna

Essa separação facilita manutenção, testes e reutilização.
