# Fila

Implementação de **fila circular** em C (FIFO: First In, First Out).

## Operações implementadas
- criar e destruir fila
- verificar se está vazia ou cheia
- enfileirar
- desenfileirar
- consultar frente
- imprimir fila

## Executar
```bash
cd /tmp/workspace/tiagofga/Estruturas-de-Dados-em-C/Fila
make
make run
```
